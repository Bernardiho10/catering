
# Component Guidelines

- **shadcn/ui**: Use the CLI to add components: `npx shadcn@latest add [component]`.
- **Styling**: Use Tailwind CSS.
- **Utils**: Use `cn()` from `@/lib/utils` to merge classes.
- **Props**: Define strict interfaces for props.
- **Accessibility**: Ensure all interactive elements are keyboard focusable and have aria-labels if icon-only.
